# Tailwind CSS 예제

## 카드 컴포넌트

여기서는 클래식 카드 컴포넌트를 보여줍니다. Tailwind CSS의 유틸리티 클래스를 사용하여 반응형 레이아웃을 빠르게 구축하며, 그림자, 둥근 모서리 및 일부 패딩 등의 스타일을 포함하여 더욱 입체감 있고 현대적으로 보이게 합니다.

<div className="my-5 max-w-md mx-auto bg-foreground dark:bg-neutral-600 rounded-xl shadow-md border p-6">
  <h3 className="text-lg font-semibold text-zinc-200 mb-2">클래식 카드</h3>
  <div className="text-zinc-200 mb-4">
    Tailwind CSS를 사용하면 반응형 카드 컴포넌트를 빠르게 구축할 수 있으며 우수한 기본 스타일과 확장성을 갖춥니다.
  </div>
  <button className="bg-blue-500 text-white px-4 py-2 rounded hover:bg-blue-600 transition">
    작업 버튼
  </button>
</div>

## 버튼 상태 데모

여기서는 기본 상태, 비활성화 상태 및 성공/경고 버튼을 포함한 여러 상태의 버튼을 보여줍니다. 모든 버튼은 Tailwind CSS의 `hover` 및 `disabled`와 같은 상호 작용 상태 클래스를 사용하여 다양한 시각 효과를 구현합니다.

<div className="my-5 flex gap-4 flex-wrap">
  <button className="bg-blue-500 text-white px-4 py-2 rounded hover:bg-blue-600 transition">기본</button>
  <button className="bg-gray-300 text-gray-700 px-4 py-2 rounded cursor-not-allowed" disabled>비활성화</button>
  <button className="bg-green-500 text-white px-4 py-2 rounded hover:bg-green-600 transition">성공</button>
  <button className="bg-red-500 text-white px-4 py-2 rounded hover:bg-red-600 transition">경고</button>
</div>

## 알림 박스 (Alert)

알림 박스는 사용자에게 중요한 정보나 경고를 알리는 컴포넌트입니다. Tailwind CSS의 배경색 및 테두리 스타일을 통해 다양한 유형의 알림 박스를 쉽게 만들 수 있습니다.

<div className="my-5 max-w-md mx-auto">
  <div className="bg-blue-100 border-l-4 border-blue-500 text-blue-700 p-4">
    <p className="font-bold">정보 알림:</p>
    <p>일반 알림 정보를 표시하는 데 적합한 정보 알림 박스입니다.</p>
  </div>

  <div className="bg-yellow-100 border-l-4 border-yellow-500 text-yellow-700 p-4 mt-4">
    <p className="font-bold">경고 알림:</p>
    <p>중요한 경고 정보를 표시하는 데 적합한 경고 알림 박스입니다.</p>
  </div>

  <div className="bg-red-100 border-l-4 border-red-500 text-red-700 p-4 mt-4">
    <p className="font-bold">오류 알림:</p>
    <p>오류 정보나 수정이 필요한 작업을 표시하는 데 적합한 오류 알림 박스입니다.</p>
  </div>
</div>

## 태그 컴포넌트 (Tags)

태그 컴포넌트는 콘텐츠의 분류나 태그를 표시하는 데 사용되어 사용자가 콘텐츠의 유형이나 주제를 빠르게 파악할 수 있도록 돕습니다.

<div className="my-5 mx-auto">
  <div className="space-x-2">
    <span className="inline-block bg-blue-100 text-blue-800 text-xs font-semibold px-2 py-1 rounded-full">태그 1</span>
    <span className="inline-block bg-green-100 text-green-800 text-xs font-semibold px-2 py-1 rounded-full">태그 2</span>
    <span className="inline-block bg-yellow-100 text-yellow-800 text-xs font-semibold px-2 py-1 rounded-full">태그 3</span>
  </div>
</div>

## 반응형 레이아웃 그리드

반응형 레이아웃 그리드는 다양한 화면 크기에 따라 표시 방식을 자동으로 조정할 수 있습니다. Tailwind CSS의 `grid` 클래스를 사용하면 데스크톱, 태블릿 및 모바일 등 다양한 장치에 적응하는 유연한 레이아웃을 편리하게 만들 수 있습니다.

<div className="my-5 mx-auto grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-4">
  <div className="bg-accent p-4 rounded shadow text-center">블록 1</div>
  <div className="bg-accent p-4 rounded shadow text-center">블록 2</div>
  <div className="bg-accent p-4 rounded shadow text-center">블록 3</div>
  <div className="bg-accent p-4 rounded shadow text-center">블록 4</div>
  <div className="bg-accent p-4 rounded shadow text-center">블록 5</div>
  <div className="bg-accent p-4 rounded shadow text-center">블록 6</div>
  <div className="bg-accent p-4 rounded shadow text-center">블록 7</div>
</div>

## 타이포그래피 스타일 (Prose)

Tailwind CSS의 Typography 플러그인(`@tailwindcss/typography`)은 우아한 타이포그래피 스타일을 제공하며 특히 Markdown 콘텐츠에 적합합니다. `prose` 클래스를 사용하면 텍스트, 목록, 인용 및 코드 블록 등의 콘텐츠가 우아한 스타일로 표시됩니다.

<div className="my-5 mx-auto prose dark:prose-invert max-w-3xl">
  <h3>지원되는 타이포그래피 콘텐츠</h3>
  <div>
    Tailwind의 Typography 플러그인을 사용하면 원래 스타일이 없는 Markdown 콘텐츠에 훌륭한 기본 타이포그래피를 적용할 수 있습니다.
  </div>

  <ol>
    <li>목록 항목 예제 1</li>
    <li>목록 항목 예제 2</li>
    <li>목록 항목 예제 3</li>
  </ol>

  <blockquote>
    해야 할 일을 하고, 나머지는 시간에 맡기세요 ~
  </blockquote>

  <div>
    인라인 코드를 삽입할 수 있습니다:
    
    예: <code>npm install tailwindcss</code>
  </div>

  <div>👉 더 많은 내용은 <a href="https://tailwindcss.com/docs" target="_blank">Tailwind 공식 문서</a>를 참조하세요.</div>
</div>
